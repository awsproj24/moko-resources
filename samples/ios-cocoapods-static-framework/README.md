# [Compose Multiplatform](https://github.com/JetBrains/compose-multiplatform) iOS CocoaPods example

This example showcases using Kotlin Multiplatform shared module in Swift as a CocoaPods with static framework.

## Build

1. publish moko-resources to local maven - `./gradlew publishToMavenLocal` in `moko-resources` root
2. build sample (in IDE or by `./gradlew build`)
